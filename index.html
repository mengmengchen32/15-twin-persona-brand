TWIN PERSONA
Brand Guidelines

Concept

The Twin Persona wordmark expresses duality through a single typographic device: solid meets outline. TWIN is filled. PERSONA is stroked. Same typeface, same weight, same size — two presentations of one identity, set side by side on a single line.

This contrast embodies the brand's core idea without decoration, illustration, or complexity. The logo does one thing well.

Primary Logo

The primary logo is monochrome. It adapts to dark and light backgrounds using only one color — white or black. This is the default for all applications.

Brand Accent Variant

For special applications (hero banners, launch pages, event materials), the outline word can carry the brand blue. The solid word stays neutral. Use sparingly — the primary logo should be monochrome in most contexts.

Brand Colors

The palette is intentionally constrained. Black and white for the logo. Electric blue as the brand accent — for buttons, links, highlights, and UI elements. Not for the primary wordmark.

The quick brown fox jumps over the lazy dog. Typography is the art and technique of arranging type to make written language legible, readable, and appealing when displayed.

Logo specs: Outfit, weight 900, letter-spacing 5px. TWIN solid, PERSONA outline, set on one line with gap equal to one character width. Outline stroke: 1.5px at 64px, scales proportionally.

Clear Space

Maintain clear space equal to the cap height of the "T" in TWIN on all sides. No other elements should enter this zone.

Animation

For digital reveals (homepage load, pitch deck intro, app splash), use the one-shot animation. Both words start as outlines, then TWIN fills to solid. Plays once and settles — never loops.

Timing: 1.8s ease-out, 0.6s delay. See the exploration file (twin-persona-logo.html) for implementation.
